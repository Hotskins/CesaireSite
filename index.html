FAQ

Comment postuler pour un stage ?

Ouvre la fiche d’une organisation, puis utilise l’email ou le téléphone indiqué pour candidater.

Est-ce que je peux filtrer par station de métro ?

Oui. Utilise le filtre « Station de métro » en haut de la page pour limiter les résultats.

Comment accéder à l’admin cachée ?

Utilise Ctrl+Shift+A puis saisis le code administrateur configuré côté navigateur.

Les nouvelles organisations sont-elles sauvegardées ?

Pour l’instant, elles restent en mémoire dans la session du navigateur (pas de base de données).
